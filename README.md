# INTELLIGENT_ROBOTIC_SORTING

Computer Vision + AI-Driven Robotic Manipulation

An intelligent robotic system that automatically sorts waste materials (plastic bottles and aluminum cans) with integration of real-time computer vision, adaptive AI decision making, and precise robotic manipulation.

🎯 Project Overview
This project implements an intelligent waste sorting system using:

Computer Vision: YOLO-based object detection

Target Objects: Plastic bottles and aluminum cans

Target Robot: Kinova Lite with 2-finger gripper

Goal: Achieve 85%+ detection accuracy and automated sorting

🏗️ System Architecture

Camera Feed → YOLO Detection → Object Classification → Grasp Point Calculation → Robot Control

🚀 Features

Computer Vision System

Real-time object detection using YOLOv8

Optimized for plastic bottles and aluminum cans

Smart bounding box generation using edge detection

FPS optimization (25-30 FPS on standard hardware)

Recording capabilities for documentation

Performance Metrics


Target Detection Accuracy: 85%+

Classes: 2 (plastic_bottle, aluminum_can)


Processing Speed: 25-30 FPS

Model: YOLOv8-small
